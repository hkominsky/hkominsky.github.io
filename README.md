# Hugh Kominsky — Personal Portfolio

Live site: [hughkominsky.com](https://hughkominsky.com)

## Overview
Personal portfolio built with vanilla HTML, CSS, and JavaScript. Features a dynamic 
project showcase, dark/light mode, and responsive design.

## Structure
- `index.html` — Home page
- `projects/` — Individual project pages and shared project styles
- `global/` — Shared styles, images, and assets used across the site
- `styles/` — Home page specific styles
- `scripts/` — All JavaScript organized by feature
- `components/` — Reusable HTML components

## Features
- Dark/light mode with localStorage persistence
- Responsive layout with mobile navigation
- Dynamic project pages driven by JS data files
- Scroll animations

## Running Locally
Open with any static file server, e.g. VS Code Live Server.
